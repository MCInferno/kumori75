# kumori75
an inabakumori-themed keyboard that uses magnets to float the PCB assembly

### Inspiration 
About a year ago, I made a hand-wired keyboard.
I don't remember why it never was completed, but it sadly never got to a stage of full functionality and has been sitting in storage, collecting dust.

Fast forward to recently, I joined the Hack Club at my school and learned about Blueprint.
I've always wanted to make something cool involving electronics, and this prompted the perfect opportunity.
I remembered my old hand-wired project, and I wanted to make something like that but just way better with grant that Blueprint provides.
I really like inabakumori and hitorie, so I started to make a keyboard themed after inabakumori (i kinda want make a hitorie/wowaka-themed keyboard some time in the future)

And that's the whole story to the beginning of this project.

### Features
- Magnetic mechanism for a bouncy typing experience
- Custom 75% Layout
- Fully 3D printed PBT Case & Polycarbonate Plate
- PCB with Hotswap Sockets and Silkscreen Art (drawn by yours truly)
- Cool Weight with inabakumori in Kanji
